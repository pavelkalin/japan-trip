# Japan Trip — March/April 2027

Family trip plan for Tokyo, Kyoto, and Osaka during cherry-blossom season.

Open `index.html` in any browser, or visit the [live site](https://pavelkalin.github.io/japan-trip/) (once GitHub Pages finishes building).

Single self-contained HTML file:

- 14-night itinerary across Tokyo (6) → Kyoto (6) → Osaka (2)
- 30+ places with multi-photo galleries (Wikipedia API), historical context, kid tips, and Google Images links for more photos
- Food guide: sushi, ramen, regional specialties, convenience stores, ekiben
- Cost estimate in USD + ILS (live exchange rates from frankfurter.dev)
- Booking timeline
